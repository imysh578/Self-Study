## 주석
- 한줄 주석: #
- 여러줄 주석: '''이 사이에 적는다''' (작은 따옴표/큰 따옴표 3번 입력)

## 자료형
### 숫자
- int
- float

#### 연산자
- 덧셈: +
- 뺼셈: -
- 곱셈: *
- 나눗셈: /
- 나머지: %
- 몫: //
- 제곱: **

#### 형변환
```python
int(3.3) # 3
float(2) # 2.0
```

#### 반올림, 올림, 내림
- 반올림: ```Round(value)``` 정수 반올림 / ```Round(value, N)``` N자리 수 반올림
- 올림: ```Math.ceil(value)```
- 내림: ```Math.floor(value)```
> 올림, 내림은 `Math`모듈 필요

### 문자
#### 연산자
- 덧셈: +로 두 문자 연결
- 곱셈: *로 문자 반복

#### index
- 첫번째 문자열 인덱스: 0
- 마지막 문자열 인덱스: -1

#### len
- 문자열 길이 출력 => len(str)
- index를 이용해서 slicing 가능 => str[0:5], str[0:len(str):2]

#### 자주 사용하는 함수들
- Lower: 소문자
- Upper: 대문자
- Find: 문자열 index
- Replace: 문자열 교체
- Endswith: 종료 문자 체크
- Split: 문자열 나누기
