## Operator  
### Fundamental Operator  
- + : addition
- - : substraction
- * : multiplication
- / : division
- % : remainder  


### bit Operator  
- & : AND
- | : OR
- ^ : XOR
- &^ : bit CLEAR


### Shift Operator  
- << : left shift
- >> : right shift

### Compare Operator
- == : same
- != : different
- >
- <
- >=
- <=  

### Assignment Operator
- =
- +=
- -=
- *=
- /=
- %=
- &=
- |=
- ^=
- <<=
- >>=  

### Increase/decrease operator
- ++
- --
** there aren't Prefix/Postfix **

### Operator Orders
1. * / % << >> & &^
2. + - | ^
3. == != < <= > >=
4. &&
5. ||
